## サイト概要
「コロナが収束したら、ぜひ行って欲しい」を紹介しあうサイト。

### サイトテーマ
コロナの時代をみんなで助け合うサイト。

### テーマを選んだ理由
コロナの影響で、潰れたりするお店が多いので、
ユーザーがなくなって欲しくないお店を紹介しあうことによって、お客をふやす目的。

### ターゲットユーザ
お店を知りたい人。
紹介したい人。
出かけたい人

### 主な利用シーン
お店を投稿する。
お店を探す。
